# iPhone Emulator Workspace

This repository currently contains the Phase 1 scaffold for an open-source iPhone-like Swift app harness.

## Workspace Layout

- `packages/swift-sdk` for the future strict-mode Swift SDK package
- `packages/runtime-host` for the future runtime host package
- `packages/diagnostics` for the future diagnostics package
- `packages/browser-renderer` for the future browser renderer package
- `packages/automation-sdk` for the future TypeScript automation SDK package
- `examples` for fixture apps and demos
- `tests` for non-SwiftPM test assets when later phases need them
