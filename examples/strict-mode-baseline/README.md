# Strict Mode Baseline Example

This example shows how the current Phase 1 `StrictModeSDK` surface can be referenced from user code.

The example is intentionally limited:

- It demonstrates symbol availability, not a working renderer or runtime.
- It does not claim SwiftUI or UIKit compatibility.
- It stays within the compile-only contracts exported today.

Use [`BaselineExampleApp.swift`](BaselineExampleApp.swift) as a shape reference for future fixture apps once later phases add real scene and view execution.
